1.1 Sejarah HTML

HTML adalah bahasa dasar untuk menampilkan halaman web pada web browser. HTML merupakan singkatan dari Hypertext Markup Language. Yang artinya adalah bahasa markup (penanda) berbasis text atau bisa disbetu sebagai formating league. HTML adalahbahaya pemograman. HTML bukanlah bahasa pemograman tetapi bahasa markup/formating.

HTML dibuat oleh Tim Berners-Lee ketika masih bekerja untuk CERN dan dipopulerkan pertama kali oleh browser Mosaic. Selama awal tahun 1990 HTML mengalami perkembagan yang sangat pesat. Setiap pengembangan HTML pasti akan menambahkan kemampuan dan fasilitas yang lebih baik dari versi sebelumnya. Sebelumnya suatu HTML disahkan sebgai suatu dokumen HTML standar, ia harus disetujui dulu oleh W3C untuk dievaluasi secara ketat. Setiap terjadi perkembangan suatu versi HTML,maka mau tak mau browser pun harus memperbaiki diri agar bisa mendukung kode-kode HTML yang baru tersebut. Sebab jika tidak, borwser tak akan bisa menampilkan HTML tersebut.

Saat ini HTML sudah mencapai versi 5 atau bisa disebut HTML5. Versi ini memiliki banyak fitur yang dapat digunakan dan memiliki beberapa tambahan penggunaan sricpt pada editor.

HTML bisa ditulis pada sebuah text editor(dreamwever, notepad++, notepad, phpstrom, brackets, netbeans, dll). Ketik kutipan coding HTML diatas pada texteditor anda dan kemudian simpan dengan format extention .html yang menandakan bahwa file tersebut adalah file HTML.

1.2 Perlengkapan Pendukung Belajar HTML

Item yang paling penting dalam melakukan pemrograman web adalah editor teks atau IDE yang mampu mendukung Anda dalam menulis kode. Sejumlah alat pendukung yang dapat dipilih mulai dari Windows Notepad yang memungkinkan Anda menulis kode file sampai Adobe Dreamweaver yang membuat kode web dalam lingkungan WYSIWYG(What You See What You Get).
